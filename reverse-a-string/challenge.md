# Reverse a String

Write a function that takes a string as input and returns it reversed.

**Rules:** You cannot use any built-in reverse methods (like `.reverse()` or `[::-1]`).

## Examples

| Input | Output |
|-------|--------|
| `"hello"` | `"olleh"` |
| `"coding club"` | `"bulc gnidoc"` |
| `"racecar"` | `"racecar"` |

## Bonus

- Handle special characters and spaces
- Make it work with Unicode (emojis! 🎉)
- Can you do it in one line?
