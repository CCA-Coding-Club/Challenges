# Reverse a String

Write a function that takes a string as input and returns it **reversed**.

**Rules:** You cannot use any built-in reverse methods or shortcuts.

## Examples

| Input | Output |
|-------|--------|
| `"hello"` | `"olleh"` |
| `"coding club"` | `"bulc gnidoc"` |
| `"racecar"` | `"racecar"` |

## Part 1: Reverse it ⭐ Easy

- Take a string as input
- Loop through it and build the reversed string character by character
- Return the result

> 💡 Think about which direction you need to iterate through the string. What index do you start at, and where do you end up?

## Part 2: Go further ⭐ Medium

Once Part 1 is working, try extending it:

- Handle special characters and spaces correctly
- Make it work with Unicode (emojis! 🎉)
- Can you do it in a single line?

## Any Language!

You can use **any programming language** you want — Python, JavaScript, C++, Java, etc.
